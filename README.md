## 💬 Support & Questions

### If App Won't Deploy:
1. Check file is named `index.html`
2. Ensure all files are in root directory
3. Wait 5 minutes for CDN
4. Hard refresh (Ctrl+Shift+Delete)

### If Features Not Working:
1. Open browser console (F12)
2. Check for error messages
3. Clear localStorage if needed
4. Test in different browser

### If Accessibility Issues:
1. Test with keyboard (Tab key)
2. Use accessibility checker (WAVE)
3. Test with screen reader
4. Check color contrast (axe DevTools)
